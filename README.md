## üìñ Usage

1. **Launch the App**: Open Auto Typer from your Applications folder.
2. **Create a New Sequence**:
   - Click the "+" button to add a new typing sequence.
   - Enter your text in the editor.
   - Set a hotkey for quick activation.
3. **Save and Test**:
   - Save your sequence.
   - Use the preview feature to test it.
4. **Activate Automation**:
   - Switch to your target application (e.g., TextEdit, VS Code).
   - Press your assigned hotkey to trigger the auto-typing.

- **Code Snippets**: Quickly insert boilerplate code in your IDE.
- **Email Templates**: Automate repetitive email responses.

## üîß Building from Source

### Prerequisites

- macOS 15.7 or later
- Xcode 17.1 or later
- Swift 5.0 or later

### Steps

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Cryptic-Warlord69/Auto-Typer-for-Macos.git
   cd Auto-Typer-for-Macos
   ```

2. **Open in Xcode**:
   ```bash
   open Typer.xcodeproj
   ```

3. **Build and Run**:
   - Select the "Typer" scheme.
   - Choose your target device (My Mac).
   - Press Cmd+R to build and run.

4. **Archive for Distribution**:
   - Go to Product > Archive in Xcode.
   - Export the app as a macOS App.
